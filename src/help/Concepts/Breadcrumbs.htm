Breadcrumbs

Language Forge is used in an Internet browser. In browsers, breadcrumbs or a breadcrumbs trail is a graphical control element that helps you know where you are in the internet site and a way to navigate to another place.

In Language Forge, the first item in the breadcrumb trail is My Projects. It is followed by the name of the current language project. The last item is the current view. Therefore, you can always learn which view you have displayed.

 

Example:

In this example, the current project is called Tests and you have the Edit view displayed.

  • You can click (My Projects) to see a list of all your projects.

  • You can click (Test) to display the List view in the current project.
