Save

In Language Forge, some changes require you to manually save your additions, deletions or other changes and some changes do not.

  • For example, if you add a new lexical entry a Save Entry button () appears.

After the changes are saved, the button label changes to read Entry Saved. Then your changes are available for any collaborators who are working in the same language project in Language Forge.

  • Other changes are saved automatically. For example, if a manager changes a person's role, then that change is saved when he or she leaves that view.
