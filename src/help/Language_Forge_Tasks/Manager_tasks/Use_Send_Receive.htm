Use Send/Receive

Managers can initiate the use of Send/Receiver to synchronize the changes done by the people who use Language Forge with the repository in Language Depot.

  1. In the project that will be synchronized, click (Send/Receive).

  2. A progress bar appears so you can see the progress. When it is finished the progress bar disappears, and the word "Synced" appears to the left of the Send/Receive button.
