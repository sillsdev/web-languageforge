Edit a comment

Do these steps to edit a comment that you previously added to a lexical entry.

  1. In the Edit Entry view, click the entry with the comment () you will edit.

  2. Click (Show Comments) to display the Comments view.

  1. In the right pane, move your mouse pointer over the comment you want to edit, and then click (Edit).

  2. Edit the comment in the text box. Click (Update).
