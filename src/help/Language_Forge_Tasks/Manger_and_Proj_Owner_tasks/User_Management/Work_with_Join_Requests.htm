Work with Join Requests

A person can submit a request to be added as a member to the currently open language project. These requests appear in the Join Requests tab.

Do these steps in the User Management view.

  1. Click the Join Requests tab.

A table appears. It has the names of any person who submitted a request.

  1. Do one of these steps:

    • To approve the request, ____ .

    • To reject the request, _____ .

  2. For each person you approved, choose a role:

    • Click the down arrow () in the Role column, and then click a role.
