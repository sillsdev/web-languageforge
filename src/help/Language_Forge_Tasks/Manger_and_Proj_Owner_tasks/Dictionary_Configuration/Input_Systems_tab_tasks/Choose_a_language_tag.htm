Choose a language tag

In Language Forge, you can choose a "language tag" and other "language identifiers" to distinguish an input system language.

In the Dictionary Configuration view, do these steps in the Input Systems tab:

  1. In the left pane, click the name of the language you will distinguish from other input system languages.

  2. In the right pane, click (More), and then choose one of the following:

    • Add IPA  for <language> if you will use the language for IPA characters.

    • Add Voice for <language> if you will record audio files with sounds.

    • Add Variant 0f <language> if you will use variants as the reason to distinguish this language.

  3. The Special box shows a result of your choice. Do one of these steps:

    • If it shows IPA Transcription, click Purpose and choose the option that indicates how this input system language will be used in this project. Then, in the Variant box, type a brief abbreviation the helps distinguish it in the user interface.

    • If it shows Voice, in the Variant box, type a brief abbreviation the helps distinguish it in the user interface.

    • If it shows Script/Region/Variant, do one of the following: Click the Script box and then click a script. Click the Region box and then click a region. In the Variant box, type a brief abbreviation the helps distinguish it in the user interface.

    • To clear a previous selection, choose none in the Special box.

  4. Click (Apply).
