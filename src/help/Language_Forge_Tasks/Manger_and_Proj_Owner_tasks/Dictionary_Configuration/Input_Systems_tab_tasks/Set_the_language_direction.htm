Set the direction of a language

In the Dictionary Configuration view, do these steps in the Input Systems tab:

  1. In the left pane, click the name of the language you will set.

  2. In the right pane, if the language is right-to-left, select () Right to left language. Otherwise, clear () this check box.

  3. If you have done all the intended changes to input writing system languages, click (Apply).
