Add new Input System

In the Dictionary Configuration view, do these steps in the Input Systems tab:

  1. Click (New).

The Select a New Input System Language dialog box appears.

  1. In the dialog box, type the language name in the Search box, and then click (Search).

    • If the input language you want does not appear in the list, click x to clear the search box. Then type a different language name.

    • If the input language you want appears in the list, click it, and then click (Add <language name>).

The dialog box closes and the new input system is added to the list below the Language Names header label which is on the left side of the Input Systems tab.

  1. Do any of the following in the right pane, as necessary:

    • In the Display Abbreviation text box, edit the abbreviation.

    • Set the language direction.

    • Choose a language tag.

  2. When you are done with the intended changes to input system languages, click (Apply).
