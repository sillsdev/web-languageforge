Add an Option Lists list item

In the Option Lists tab, you see various lists, such as Part of Speech. You can add items to these lists.

In the Dictionary Configuration view, do these steps in the Options Lists tab:

  1. In the left pane, click the desired list.

  2. In the right pane, click the text box and type the name of the list item.

  3. Click (Apply).
