Set up Gather Words from Semantic Domains

In the Dictionary Configuration view, do these steps in the Tasks tab:

  1. In the left pane, click From Semantic Domains.

  2. In the right pane, do these steps:

    • Click the Language down arrow () and then click a language.

    • Select () each field that you want to contain information, in addition to the Word field as people gather information.

    • Click (Apply).
