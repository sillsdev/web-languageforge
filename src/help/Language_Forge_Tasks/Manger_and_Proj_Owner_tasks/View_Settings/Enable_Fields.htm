Enable Fields

Do these steps in the View Settings view. This view shows a tab for each role.

  1. Click the tab for the role you will change.

  2. If the pane that shows fields is not shown, click the Enable Field for <role> header.

  3. In the left pane, click the desired field.

    • If you cannot see the field, click (Show All Fields).

This symbol appears for fields that are currently enabled: .

  1. In the right pane, do these steps:

    • Select () Show field if you want people in this role to see the field. Otherwise, clear () it.

    • Select () Override Input Systems if you want to change which input systems the people can see, or the order they appear.

If you selected it, do these steps:

      • Select () or clear () input systems as desired.

      • Click an input system, and then click the (up arrow) or (down arrow) to change the order of the input systems.

  1. Repeat any of the above steps for different roles or different fields.
